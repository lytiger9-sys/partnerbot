이건 판매용 프리미엄 고급 상품이기 떄문에 개발자 id를 환경 변수 파일에 두고, 그 아이디로 봇에게 dm으로 보내면 현재 봇이 초대되어있는 서버와 라이센스, 남은 유효기간을 보여주는 임베드를 보낸다. dm으로 개발자인 내가 라이센스 생성 명령어를 사용하면 며칠동안 사용할 수 있을지 입력하는 선택창을 만들고, 입력하면 그 일수만큼 작동하는 라이센스를 만든다.

일단 이게 판매용 기능에 대한 기초임.

그러면 봇은 서버에 초대되면, 내가 봇을 구매한 사람에게 라이센스를 넘겨주고, 라이센스 등록 명령어를 통해 서버 관리자가 봇을 사용할 수 있게 한다.

파트너에 대한 기능은 일단 서버 관리자가 라이센스를 등록하면, 최소 인원 수와 봇 비율, 제휴명, 로그 채널, 파트너 생성시 카테고리를 설정할 수 있는 /설정 명령어를 통해 실행하고
/임베드 게시를 통해 해당 조건을 명시하는 임베드를 게시한다.(최신 스타일 discord component v2) 임베드에는 버튼이 두개 있는데, 하나는 파트너 신청/조건 그리고 다른 하나는 웹훅 입력이다. 
    임베드에 명시할 내용은 제목으로 (서버명)의 파트너 봇입니다. 임베드 내용으로 1. 최소 인원 수, 2. 제휴명, 3. 1일 1홍보
/홍보글 설정을 통해 어떤 메세지를 웹훅으로 보낼지 해당 채팅 채널에서 선택할 수 있다.(해당 채팅 채널의 가장 오래된 메시지로 설정)

각 버튼에 대한 설명

파트너 신청/조건: 파트너를 요청한 사람이 이 버튼을 누르면 서버 링크를 제출할 수 있는 창이 뜨고, 입력하면, 봇이 파트너로그채널에 해당 서버 링크와 유저명을 올리고, 최소 인원 수, 봇 비율을 체크하여 조건에 맞는지 확인 후 ephimeral메시지로 조건 성립 여부에 대해 알려주고 해당 조건에 만족했다면, 제휴명으로 자신의 서버에서 파트너 채널을 만들고 웹훅을 준비하라는 안내문을 추가로 보내준다.(ephimeral메시지)

웹훅 입력: 만약 앞의 버튼으로 아무것도 제출하지 않은 유저(로그 채널에서 확인)라면 파트너 신청/조건을 먼저 해달라는 ephimeral메시지를 보내고, 이미 제출한 유저라면, 웹훅 및 제휴명을 입력할 모달창을 연다.
모든걸 입력하면, 일단 웹훅으로 정해놓은 홍보글을 보내고, 보내지지 않는다면 파트너 채널 개설하지 않기, 보내진다면, 지정된 카테고리에 파트너 채널 생성(제휴명은 모달창에서 받은 그대로)
홍보 주기는 무조건 하루에 한번, 오후 3시이며, 웹훅 검사 주기도 메시지를 보낼 때 오류가 생기는지만 확인하면 된다. 

만약 봇이 만든 파트너 채널에 하루에 한개 이상의 메시지가 올라오면, 해당 유저를 3일동안 타임아웃시킨다.



모든 명령어는 라이센스를 입력한 서버에서만 작동하게 해야 한다.
